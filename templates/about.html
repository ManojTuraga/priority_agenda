Priority Agenda

This program utilizes a Node-Based Binary Search Tree to house accounts that are created for this site. Each account contains a List-Based Maxheap that will present the most imporant item to complete. The most important item is calculated by comparing the due dates, then the date the task was created, and a user given "weight" for the task in that order.
